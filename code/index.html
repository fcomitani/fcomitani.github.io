A list of packages published on PyPI or on GitHub

  • tapir


    Transcriptome analysis in Python. This library provides functions for the analysis and plotting of gene expression data. It imports differential expression functionalities from EdgeR to Python.

  • miniW2V

    Minimal word2vec skip-gram implementation in PyTorch with negative sampling. It includes a web scraper for scientific abstracts from PubMed, Google Scholar and bioRxiv.

  • SimpSOM


    A simple and easy to learn library for Kohonen Self-organizing maps.
